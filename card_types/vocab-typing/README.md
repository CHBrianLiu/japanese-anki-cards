# Vocab Typing Card Type

## Purpose
This card type tests your ability to produce the correct form of a Japanese word within a context sentence.

## Required Fields
- **Sentence**: Example sentence with `[]` marking the missing word.
- **TargetWord**: The correct form to type.
- **Hint**: English clue to identify the word.
- **Reading**: Furigana reading for the back side.

## How to use
**Front**: Read the sentence and the hint, then type the correct Japanese word into the box.
**Back**: See if you were correct. The back reveals the reading, dictionary form, and pitch accent.
