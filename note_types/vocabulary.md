# Vocabulary Note Type

This note type is designed for comprehensive Japanese vocabulary learning, covering both recognition and active production (typing).

## Fields

1. **Word**: The dictionary form of the word (e.g., 食べる).
2. **Reading**: The word with furigana in `Kanji[Reading]` format (e.g., 食[た]べる).
3. **Sentence**: An example sentence with `[]` marking the target word position.
4. **TargetWord**: The specific form of the word used in the sentence (the conjugated answer).
5. **Hint**: An English clue or meaning to help with the typing task.
6. **Pitch**: Pitch accent information (e.g., [2]).

## Generated Card Types

This note type generates two cards:

### 1. Recognition
- **Front**: Displays the dictionary form (**Word**).
- **Back**: Shows the reading, pitch, and context sentence.
- **Goal**: Test if you can recognize the word and recall its reading/meaning.

### 2. Vocab-typing
- **Front**: Shows the **Sentence** with a hint. You must type the **TargetWord**.
- **Back**: Shows the correct word, reading, pitch, and the full sentence.
- **Goal**: Test if you can produce the correct form of the word in context.
