# Japanese Anki Cards Repository

A collection of Anki cards for Japanese language learning.

## Structure

- `templates/` - Card/note templates to copy from
- `notes/` - Ready-to-import card files
- `resources/` - Images, audio, etc.
- `docs/` - Study guides and notes

## Usage

1. Copy a template from `templates/` to `notes/` when creating new cards
2. Import `notes/*.txt` or `notes/*.csv` into Anki
